# AETHER - Real Estate Website

A modern, high-performance real estate platform built to showcase property listings and neighborhood guides. This project was built with the help of [Relume](https://library.relume.io/) and exported to React.

## 📖 Documentation
For an in-depth look at what AETHER offers, its features, and tech stack architecture, please read the [Documentation](./documentation.md).

## 🚀 Tech Stack
- **Framework:** Next.js (App Router)
- **Library:** React 19
- **Styling:** Tailwind CSS 4
- **UI Components:** Relume UI, Framer Motion, Lucide React

## 💻 Getting Started

First, install the dependencies:
```bash
npm install
```

Then, run the development server:
```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## 🛠️ Project Structure
- `src/app/` - Page routes (Home, About Us, Contact, Properties, Neighborhood Guides).
- `src/components/` - Reusable UI sections built via Relume (Hero, Galleries, Navigation, etc.).
- `src/data/` - Static mock properties data.
- `public/` - Static image assets and icons.
